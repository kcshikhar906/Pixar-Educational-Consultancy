# **App Name**: Pixar Educational Consultancy

## Core Features:

- Engaging Homepage: Landing page with clear CTAs, service overviews, and testimonials.
- Comprehensive 'About Us': Dedicated pages detailing our mission, team, and certifications.
- Service Details: Detailed descriptions of our documentation, guidance, and assistance services.
- Country Guides: Informative pages for each supported country (Europe, Australia, USA, New Zealand) that contain a table of top universities and basic facts.
- Smart English Test Advisor: Interactive questionnaire to recommend suitable English proficiency tests (IELTS, PTE, TOEFL, Duolingo, etc.) that uses a reasoning tool to ensure relevancy to the user's individual level, timeline and budget.
- Pathway Planner: System to suggest countries and universities that best match a student's education level and desired field of study. The system uses a reasoning tool to decide when/if the selected school has any relevance to their study area or education.
- Easy Contact Options: Contact form, Google Map integration, and direct chat link.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and credibility.
- Background color: Light blue (#E8EAF6), creating a calm and trustworthy backdrop.
- Accent color: Soft violet (#9575CD) to highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, objective feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use professional, consistent icons across the website. Icons to support a user's understanding of the application requirements.
- Use clear and logical layouts for each page to guide the user effectively. Use tables for presentation of multiple options to enable rapid comparisons.
